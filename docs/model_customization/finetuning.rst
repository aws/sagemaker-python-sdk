Fine-tuning
===========

Train models on your labeled data to improve performance on specific tasks using SFT, DPO, RLAIF, and RLVR techniques.

**What you'll accomplish:**
- Configure training jobs with the v3 SDK
- Monitor training progress and metrics
- Create fine-tuned model packages for deployment

**Choose your technique:** SFT for general fine-tuning, DPO for preference learning, RLAIF/RLVR for reinforcement learning approaches.
