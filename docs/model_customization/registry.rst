Dataset and Evaluator Assets
=============================

Manage datasets, evaluators, and model artifacts through SageMaker's AI Registry for organized model development workflows.

**What you'll accomplish:**
- Register and version training datasets
- Create custom evaluators and reward functions
- Track model lineage and metadata

**Prerequisites:** SageMaker permissions, prepared datasets, understanding of evaluation metrics

**Use cases:** Dataset versioning, custom evaluation workflows, model governance and compliance tracking.
